github账户的注册和配置

  • Github注册：https://github.com/
  • 创建代码库：
  • 在Repository name下填写yourname.github.io，Description (optional)下填写一些简单的描述（不写也没有关系
  • 代码库设置:Setting
    • 接下来开启gh-pages功能，点击界面右侧的Settings，你将会打开这个库的setting页面，向下拖动，直到看见GitHub Pages
